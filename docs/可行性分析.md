# 可行性分析

## 难点 

1. 按用户喜好推荐，需要综合进行分析。

2. UI设计与风格统一

3. 动画设计



## 解决方案

1. 壁纸图片源：
  
   - Unsplash等免费图片源
2. 图片推荐流程：
   - Unsplash提供图片collection，直接从一个collection中加载图片，省去了图片服务端的开发。
   - 使用unsplash用户建立多个collection，可以按类别推荐图片。
3. 数据库设计采用Room进行设计
4. 图片列表使用RecyclerView
